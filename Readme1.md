## 1. 개 요 (개요라는 말을 바꿀수도 있음)
**작성방향 : 왜 이 문제를 해결하고자 하는가? 이 문제를 해결하는 것이 어떠한 의미가 있는가 무엇을 확인해보고 싶은가?**

영화는 가장 전형적이고 보편적인 문화활동 중 하나로, 가장 영향력있는 콘텐츠 중 하나이다.
디지털 시대에 접어들어서는 다양한 온라인 영화 리뷰 플랫폼이 존재하고, 이 리뷰의 평가가 영화의 흥행과도
직결되는 척도로 활용되고 있다[[1]](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART002021510)

이번 프로젝트에서는 SF영화의 상징인 스타워즈 시리즈에 대한 리뷰를 분석해보고자 한다.
스타워즈는 ~~ 언제 나왔고, 얼마나 성공하였으며 ~~ 등등의 통계자료로 근거를 마련합니다.

스타워즈 시리즈는 항상 성공만 했던 것은 아니다. 시리즈가 지속될수록 넓어진 세계관과 무리한 복선의 설정은
관객들로 하여금 수많은 추측을 부여함과 동시에 그에 따른 기대가 커져있는 상황이다. 따라서 이번
프로젝트에서는 스타워즈 시리즈별 리뷰 데이터를 직접 수집하여 긍정, 부정, 중립 등을 분류하고 예측하여
부정적인 리뷰가 높은 영화에 이슈를 분석해보고자 한다.

## 2. 데이터
### 2.1 데이터 수집 (필요시 추가)
 - case1: 직접 수집한경우
   - 어디서 얼마나 어떻게(소스코드를 저장소에 추가) 수집했는지 자세하게
   - 데이터가 어떤 항목을 갖는지(예시 - 날짜, url, 별점, 리뷰텍스트 등)
   - 데이터의 총 건수, 수집대상 기간 등에 대한 개괄적인 정보 언급
 - case2  : 있는 데이를 활용한 경우
   - 데이터에 대한 출처 (총 건수, 수집 대상 기간 등등)
   - 데이터가 어떠한 항목으로 구성되고, 라벨이 있는지 여부를 정확히 확인
   - 데이터 라벨에 대한 품질을 간단하게 확인해보기 (랜덤으로 50개 정도 확인)
### 2.2 탐색적 데이터 분석
 - 있는 그대로의 데이터를 표 / 차트의 형태로 표현한다 (기술통계학)
 - 표는 마크다운에서 제공하는 기능을 활용한다
   
|구분|일시|그룹|리뷰텍스트|라벨|
|---|---|---|---|---|
|Season 1| 1999.04.|~~~|good|1|
 - 그림의 경우에는 matplotlib/seaborn 등을 적절히 활용하되 적당한 크기로 잘 조절하여 표현해야 가독성이 높아진다
 - 표로 작성할 항목은 샘플 데이터, 데이터 라벨링 기준 등 세부적인 정보를 전달할 필요가 있을때
 - 그림으로 작성할 예시 항목은 문장 길이의 분포, 라벨의 분포, 일자별 분포(일자 데이터가 있다면)

## 3. 학습 데이터 구축
**총 데이터가 5만건이라고 한다면 이중에서 10%정도 혹은 2,000~ 3,000건 정도의 데이터를 학습데이터로 만든다.
학습 데이터를 추출할 때 단순한 랜덤이 아닌 데이터의 분포를 보고 판단한다. 예를 들어 부정데이터(특정 라벨의 데이터)
의 비중이 매우 작을 때는 부정과 긍정을 1:1로 구축할 수 있다. 혹은 일자 별(특정한 기준-attribute)로 
비율을 바탕으로 학습데이터를 구축할 수 있다.**
### 3.1 데이터 라벨링
 - case1: 직접 수집한 경우
   - 데이터 라벨링을 어떠한 기준으로 했는지 (긍정/부정/중립/관련없음 등)
   - 필요시 Gemini Pro 도구 활용 가능
   - 라벨링 결과에 따른 라벨링 데이터 분포 시각화
 - case2: 있는 데이터를 활용한 경우
   - 데이터 라벨에 대한 자세한 검증
   - 라벨이 제대로 부여되지 않은 경우에 대한 재라벨링 (Gemini 활용 가능)
   - 마찬가지로 시각화


### 3.2 학습 데이터 
최종 학습데이터에 대한 분포를 탐색적 데이터 분석 형태로 기술한다. (표 / 차트 포함)

## 4. MobileBERT 모델 학습 (Fine-tuning)
 - 학습 : 검증은 8:2 혹은 7:3으로 설정
 - MobileBERT를 학습시키고 1. traning loss, 2. training/validation accuracy에 대한 수치적인 정보와
차트를 그린다.
 - 모든 데이터가 라벨링이 되어 있는 경우에는 학습된 모델로 inference하여 전체 대이터셋에 대한
accuracy를 보여준다

## 5. 문제제기에 대한 결과(악평 스타워즈에 대한 원인 분석)

- 차트를 통해 긍부정(문장분류에 따른 분류의 비중 등) 흐름 표현 및 분석
